# EPDŘ – Czech Driver Worktime Reference (v1.0.0)

This directory contains **EPDŘ reference data**, released as part of
the `epdr-v1.0.0` release.

## What this is
- Reference spreadsheet for Czech driver worktime records
- Legal context: Czech Labour Code (Act No. 262/2006 Coll.)
- Intended as **data / reference input**, not ODWS specification

## What this is NOT
- Not ODWS normative documentation
- Not translated documentation
- Not part of the ODWS core specification

Kept for historical and reference purposes.
