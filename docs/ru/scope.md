# Объем проекта

Этот документ определяет область применения
проекта Open Driver Worktime Space (ODWS).

Его цель – четко указать
чему посвящен проект
и, столь же ясно, чего он не делает.

---

## 1. Что охватывает проект

ODWS занимается вычислениями
и прозрачное описание:

- рабочее время водителя
- время включено и не включено в работу
- основные права, связанные с работой
получено из отработанного рабочего времени

Проект фокусируется на:

- **реальные эксплуатационные данные**
- **детерминированные вычисления**
- **первое использование в автономном режиме**
- **проверяемые и воспроизводимые результаты**

Вывод системы всегда состоит из:
- вычисленный результат
- объяснение того, как был получен результат
- идентификация используемой интерпретационной модели

---

## 2. Что проект намеренно не охватывает

ОД намеренно:

- не заменяет юридическую консультацию
- не является авторитетным толкованием закона
- не разрешает споры
- не несет ответственности за окончательные решения по заработной плате
- не заменяет системы бухгалтерского учета и расчета заработной платы

Проект:
- **вычисляет**
- **не обеспечивает исполнения решений**

---

## 3. Первая стабильная область применения (этап 1)

Первая стабильная фаза проекта ограничивается:

- обработка рабочего времени водителя
- основные категории времени (работа, доступность, отдых)
- локальная обработка данных без зависимости от облака
- справочные результаты в машиночитаемых форматах

Этот этап НЕ направлен на обеспечение:

- полный расчет заработной платы
- интеграция с бухгалтерскими системами
- всестороннее законодательное освещение
- автоматизированная связь с государственными органами

---

## 4. Принципы расширения

Объем проекта может быть расширен только за счет:

- явное решение
- сохранение простоты
- без нарушения основного детерминизма

Любое расширение ДОЛЖНО:
- быть отделимым
- быть необязательным
- не нарушать существующие вычисления

ODWS явно включает
детерминированные вычисления по требованию
как часть своей нормативной сферы.

Системы, которые только вычисляют результаты
после юридического или административного оформления
выходят за рамки ODWS.
