# Portée du projet

Ce document définit le champ d'application
du projet Open Driver Worktime Space (ODWS).

Son objectif est de préciser clairement
ce que le projet aborde
et, tout aussi clairement, ce qu'il ne fait pas.

---

## 1. Ce que couvre le projet

ODWS aborde le calcul
et description transparente de :

- temps de travail du chauffeur
- temps inclus et exclus du travail
- les droits de base liés au travail
dérivé du temps de travail effectué

Le projet se concentre sur :

- **données opérationnelles réelles**
- **calcul déterministe**
- **Utilisation hors ligne d'abord**
- **résultats vérifiables et reproductibles**

La sortie du système consiste toujours en :
- un résultat calculé
- une explication de la manière dont le résultat a été obtenu
- identification du modèle interprétatif utilisé

---

## 2. Ce que le projet ne couvre pas intentionnellement

ODES intentionnellement :

- ne remplace pas un avis juridique
- n'est pas une interprétation faisant autorité de la loi
- ne résout pas les litiges
- n'assume pas la responsabilité des décisions salariales finales
- ne remplace pas les systèmes de comptabilité ou de paie

Le projet :
- **calcule**
- **n'applique pas les décisions**

---

## 3. Première portée stable (Phase 1)

La première phase stable du projet se limite à :

- traitement du temps de travail des chauffeurs
- catégories de temps de base (travail, disponibilité, repos)
- traitement des données local sans dépendance au cloud
- sorties de référence dans des formats lisibles par machine

Cette phase n’a PAS pour objectif de fournir :

- calculs de paie complets
- intégrations de systèmes comptables
- une couverture législative complète
- communication automatisée avec les autorités publiques

---

## 4. Principes d'extension

La portée du projet ne peut être étendue que par :

- décision explicite
- préservation de la simplicité
- sans rompre le déterminisme de base

Toute extension DOIT :
- être séparable
- être facultatif
- ne pas perturber les calculs existants

ODWS inclut explicitement
calcul déterministe à la demande
dans le cadre de son champ d’application normatif.

Des systèmes qui calculent uniquement les résultats
après finalisation juridique ou administrative
sont en dehors du champ d’application de l’ODWS.
